# Proyecto TDD Chatbot

## Descripción
Este proyecto es un chatbot que responde automáticamente a preguntas comunes utilizando Test-Driven Development (TDD). La idea es crear un software robusto mediante la escritura de pruebas antes de desarrollar las funcionalidades del chatbot.

## Tecnología
- JavaScript 
- Node.js 
- Express 
- Mocha/Chai para pruebas

## Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu_usuario/proyecto-tdd-chatbot.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd proyecto-tdd-chatbot
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```

## Uso
1. Ejecuta el servidor:
   ```bash
   npm start
   ```
2. Accede al chatbot en tu navegador o cliente API favorito.

## Pruebas
Para correr las pruebas, utiliza:
```bash
npm test
```
